Let's Do It is task management app (a to-do list) designed for studying purposes. This app is implemented using the AngularJS framework and Angular Material

In order to work properly, this app interacts with a API to read, create, update e delete tasks. But don't worry, we already implemented one (a RESTFul Web API using NodeJS and MongoDB) and it is available and working well

Well, I'm sure you came here to test the APP and see how it works. If you want to create a task, just click in the button on the bottom right corner and a form will pop up.

The Lets Do It App provides all the basic operations regarding tasks (read, creation, editing and deleting). Some other operations were added, such as prioritize and mark the task as done. The tasks are separated as follows:

assignment TASKS (unfinished or pending tasks) - can be edited, prioritized, marked as done and deleted. assignment_turned_in FINISHED (finisehd or done tasks) - can be viewed and deleted.

Feel free to use this APP the way you like. If you want to report any issue, go to the APP repository on GitHub.
