"Let's Do It" is a task management app (a to-do list) designed for studying purposes. This app was implemented using the following:

 AngularJS framework

An open-source, JavaScript framework for front-end web applications development. It simplifies both development and testing.

 Angular Material

A UI Component framework for AngularJS developers, based on Google's Material Design Specification.

In order to work properly, this app interacts with an API to read, create, update e delete tasks. But don't worry, we already implemented one (a RESTFul Web API using NodeJS and MongoDB) and it is available for you!

Well, I'm sure you came here to test the APP and see how it works. If you want to create a task, just click on the "add" button at the bottom right and a form will pop up.

The Let's Do It App provides all the basic operations regarding tasks (read, creation, editing and deleting). Some other operations were added, such as prioritize and mark the task as finished or unfinished. The tasks are separated as follows:

assignment TASKS

These are unfinished or pending tasks. They can be edited, prioritized, marked as done and deleted.

assignment_turned_in FINISHED

These are finished or completed tasks. They can be viewed, deleted and reopened (marked as unfinished).

Finished tasks are pretty intuitive by themselves. On the other hand, unfinished tasks have some explicit information on them, priority and deadline to be more precise. They are showed as follows:

X days  DEADLINE

These are named "chips" and they show the remaining days until a task reaches its deadline, if it exists. The colors give a visual hint about how close to its deadline a task is, ranging from green (far) to red (close).

star PRIORITY

The star symbol is used to indicate a task of critical priority.

Also, you will notice that the unfinished tasks are ordered using the two characteristics above, both "priority" and "deadline". We decided, arbitrarily, that "priority" would have a higher...well...a higher priority than "deadline". Haha.
